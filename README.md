# MathCampRSA

Contents:

prime_number_generation.py - Fermat primality testing, modExp algorithm, put-together function to generate primes

generate_keypair.py - Contains extended euclidean algorithm, generates keypair with generated primes. Writes keys to files.

encrypt_decrypt_files.py - Reads in message file and key file, and will encrypt or decrypt, writing to an output file.
